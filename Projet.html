      SAé sur le réseau

Nous avons donc eu des question de possé pour nous apprendre à utiliser différents outils réseau.       SAé traitement des données

    Dans cette SAé nous devions extraire des données en utilisant un programme basé sur le langage de programation python et retranscrire les résulta sur une page html. Pour monn groupe, nous devions à partir de trame GPS resortir les différente positions des trame, le départ, l'arrivé, l'altitude les point cardino, la distance, la vitesse et le point le plus élevé. Pour réaliser cette tache nous avons du utiliser des outils de gestion de projet pour organiser, répartir et simplifier la communication de l'équipe, Pour cela nous avons utilisé GIT. Deplus nous avons dû représenter clairement le cahier des charge et pour cela nous avons utilisé umbrello.
